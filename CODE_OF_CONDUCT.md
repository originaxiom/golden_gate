# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.

In short: be respectful and constructive. Harassment, discrimination, and personal attacks are not
tolerated. We care about the work and about the people doing it.

## Scope

This applies to all project spaces (issues, pull requests, discussions) and to public spaces when an
individual is representing the project.

## Enforcement

Report unacceptable behavior by opening an issue or contacting the maintainer via the repository.
Reports are handled confidentially and in good faith. Maintainers may remove, edit, or reject
contributions and comments that violate this code, and may ban contributors for serious or repeated
violations.

The full text of the Contributor Covenant 2.1 is at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
